# GLIV

A universal graphics library for monochrome displays with flexible functionality and its own toolset.

Static memory allocation. Designed for comfortable use on embedded systems.

Use the gliv_windows_minifb_example target to experiment with the library.

You can debug your image code in the example using a gliv work block or animations in an infinite mfb_wait_sync loop. If you need a more flexible debugging system on a PC, let us know, and we'll figure out a solution.

Some algorithms were borrowed from [Adafruit-GFX-Library](https://github.com/adafruit/Adafruit-gfx-library).
